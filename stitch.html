Stitch & Finalise

Combine the composer's Track Sheet with their artist/contributor credits, add engineers, and export the upload sheet.

1Import the completed Track Sheet (.xlsx)

The sheet the composer filled in (columns A–I). Reads the Upload Sheet tab.

2Import the composer's credits (.csv)

The file exported from the Composer Submission Tool.

3Match preview & imported credits

Album-wide credits apply to every track; per-track exceptions are matched below. You can edit the imported artists and contributors here before exporting.

Import both files to see matches.

4Add engineers (album-wide)

Search the contact list — matches (with IDs) appear as you type so you can catch close names. Applied to every track.

Engineer exceptions (optional)

If some tracks have a different mixing or mastering engineer, add an exception, tick every track it applies to, and set the engineer(s). Leave a field blank to keep the album-wide engineer for that role.

5Combined preview

One row per track, in the original sheet order. Check it, then export.

Import both files to preview.

Import the Track Sheet and the credits file to begin.
